# Support

## Getting Help

- **Documentation**: See the [README](./README.md) for usage and API reference
- **Issues**: [Open a GitHub issue](https://github.com/mxn2020/geenius-ui/issues) for bugs or feature requests
- **Discussions**: Use [GitHub Discussions](https://github.com/mxn2020/geenius-ui/discussions) for questions and community support
- **Email**: support@geenius.app for private inquiries

## Response Times

- **Bug reports**: Within 48 hours
- **Feature requests**: Reviewed weekly
- **Security issues**: Within 24 hours (see [SECURITY.md](./SECURITY.md))
