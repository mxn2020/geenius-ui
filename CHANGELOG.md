# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.4] - 2025-03-03

### Fixed
- Fix workspace:* dependency resolution for pnpm publish
- Add overrides documentation for consuming projects

## [0.1.3] - 2025-03-03

### Added
- Vanilla CSS variants: `@geenius-ui/react-css` and `@geenius-ui/solid-css`
- 47 React + 44 Solid components across both styling variants

### Fixed
- Export all compound component sub-parts

## [0.1.2] - 2025-03-02

### Added
- AlertDialog compound component pattern (Tailwind variant)
- Select with compound sub-components (Tailwind variant)

## [0.1.1] - 2025-03-02

### Fixed
- Missing peer dependencies declarations
- Build output paths

## [0.1.0] - 2025-03-02

### Added
- Initial release
- Core components: Button, Card, Input, Textarea, Label, Badge, Alert, Loading
- React (Tailwind) and Solid (Tailwind) packages
- Shared types package
